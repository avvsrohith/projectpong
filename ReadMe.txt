PONG Classic arcade Game

->Minimalistic theme and UI
->Switches for controlling various conditions
->Darkmode

->Consists of 3 powerups Long bat, Large ball and an extra life(Twice as rare)
->For every 5 bounces with player's bat, there is a 50 percent chance for a powerup to appear

->Consists of CPU mode
->In easy, CPU is slower than ball and hence eventually loses if it cannot catch the ball
->In hard, CPU is invincible and has no chance to lose

->Sound effects and vibrations added for various conditions and buttons with the ability to disable
